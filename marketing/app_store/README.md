# App Store Screenshot Kit

This folder generates premium 6.5-inch App Store screenshots from raw app captures.

## What to do

1. Put your raw screenshots in [`input/`](/Users/serhat/Desktop/mobile/marketing/app_store/input) using the exact filenames listed in [`slides.json`](/Users/serhat/Desktop/mobile/marketing/app_store/slides.json).
2. Run:

```bash
swift /Users/serhat/Desktop/mobile/marketing/app_store/generate.swift
```

3. Upload the PNG files from [`output/`](/Users/serhat/Desktop/mobile/marketing/app_store/output) to App Store Connect.

## Current concept

- Premium ivory background
- Soft editorial gradients
- Clean dark typography
- Warm gold and indigo accents
- Screenshot-led layouts with luxury spacing

## Output size

- 1242 x 2688

This is valid for the iPhone 6.5" App Store screenshot slot.
